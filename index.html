How did I start programming?!

I began programming by making games using C# in Unity Engine at 12. A few years later, to find the field I'm the most interested in, tried different areas such as Computer Hardware (CompTIA A plus course), Computer Networks (CompTIA Network plus, and a few lessons in Persian), Android development (using the Dart language and the Flutter framework using Android Studio and B4a), the Windows form applications development (using Visual Studio, Microsoft SQL Server and C#) and finally learning other programming languages like Python. Meanwhile, I learned Data structures, algorithms, and C++ for participating in the student Computer Olympiads. Finally, at the end of the day tried FrontEnd web development and liked that the most. So I chose this field to continue the journey!
